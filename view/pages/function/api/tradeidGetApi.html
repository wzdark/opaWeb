1.接口说明

获取交易号

2.使用场景

post请求需提供交易号 可以防止重复提交。

3.接口调用说明

3.1请求说明

url https://openapi.yonyouup.com/system/tradeid
格式 JSON
https请求方式 GET


3.2输入参数说明

参数 是否必填 描述
from_account 是 调用方id
appid 是 api编号
secret 是 密钥

3.3请求示例

https://openapi.yonyouup.com/api/system/tradeid?
from_account=FromAccount&
appid=APPID&
secret=APPSECRET


3.4返回参数说明

参数说明 描述
errcode 错误码，0 为正常。
errmsg 错误信息。
id 交易号。
createDate 创建时间。
status 状态( init 初始化 running 运行中 finished 已完成)。

3.5正确返回示例

JSON示例:

Content-type: text/html; charset=utf-8
{
"errcode": "0",
"errmsg": "success",
"trade": {
  "createDate": 1427332810639,
  "id": "40d8c9b4d35611e4ab8e0017fa000443",
  "status": "init"
}
}

3.6错误返回示例

Content-type: text/html; charset=utf-8
{
"errcode" : "2001",
"errmsg" : "fail"
}
